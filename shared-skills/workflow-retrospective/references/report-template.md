# Retrospective Report Template

회고 리포트를 아래 구조로 생성한다.
`workflow-retrospective/metrics/retrospective_{YYYY-MM-DD}.md` 에 저장.

---

## 리포트 생성 지침

리포트를 작성할 때 지키는 원칙:

**데이터 우선:** 모든 주장에 수치를 붙인다. "좀 느린 것 같다" 대신 "평균 45초, 이 중 62%가 browser_start".

**비교 제공:** 가능하면 이전 기간과 비교한다. "지난주 대비 평균 소요 시간 12% 감소".

**실행 가능한 제안:** "개선이 필요하다" 대신 "test-pc-worker/references/windows-commands.md의 check-block 섹션에서 Start-Process chrome 전에 기존 Chrome 프로세스 확인 로직을 추가하면 cold start를 피할 수 있다".

**간결함:** 핵심만. 이미 잘 돌아가는 부분은 길게 다루지 않는다.
