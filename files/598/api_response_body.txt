=== Gamma AI Generation API SSE Response Capture ===
Endpoint: https://ai.api.gamma.app/ai/v2/generation
Method: POST
Status: 200 OK
Format: Server-Sent Events (SSE) — `event:` + `id:` + `data:` framing per chunk
Prompt: "What is the capital of South Korea?"

KEY EVENTS:
- event: content (id 1..12)  — streaming text chunks (Korean outline content; encoding shown as UTF-8 mojibake in clipboard)
- event: done   (id 13, data: stop)  — COMPLETION SIGNAL

Each event has form:
    event: <name>
    id: <int>
    data: <text>
    [optional additional `data:` continuation lines]
    <blank line terminator>

Multi-line `data:` lines are concatenated by SSE spec to form the chunk content.
Completion is signalled by `event: done` with `data: stop`.
